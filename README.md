# imagens-pokemon

Esse repositório serve somente para eu colocar imagens de pokémons que não tem uma boa resolução no PokémonDB e para colocar alguns fakemons, ele serve para meu RPG de Mesa de pokémon, que lê o nome e identifica qual imagem deve ser carregada.
